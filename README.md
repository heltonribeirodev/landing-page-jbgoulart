## 🧠 Sobre o Projeto

Projeto desenvolvido para fortalecer a presença digital de um eletricista profissional, apresentando seus serviços de forma clara, estratégica e persuasiva.

A página foi estruturada com foco em conversão e destaque para os principais serviços oferecidos.

O objetivo é transformar visitantes em contatos diretos via WhatsApp.

## ⚙️ Funcionalidades

- ✅ Apresentação dos serviços elétricos
- ✅ Seção de diferenciais e credibilidade
- ✅ Botão direto para WhatsApp
- ✅ Formulário para captação de leads
- ✅ Layout 100% responsivo
- ✅ Estrutura otimizada para SEO
